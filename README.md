# 在庫管理サイト

シンプルなブラウザ版の在庫管理ツールです。以下を記録できます。

•	品名 / メーカー / 棚番号 / 最低数 / メモ
•	在庫カウント（入庫・出庫・棚卸し）
•	入力履歴（出庫 / 入庫 / 棚卸し（実数セット） / 品目登録を分離）


## 画面構成（タブ）

1. 現在の在庫（在庫操作 + 在庫一覧）
　o	備品の在庫記録を入力できます
  <br>
　o	記入者名を入力することで、誰が操作をしたか記録できます
  <br>
　o	現在の在庫を一覧で確認できます
  <br>
　o	在庫一覧は棚番号が若い順で表示されます
  <br>
　o	在庫一覧で登録した情報を編集できます
  <br>
　o	在庫が最低数を下回った場合は、対象のものが赤文字になります
  <br>
3. 品目登録
  <br>
　o	新しく購入したものを登録できます
  <br>
　o	記入者名 / 品名 / メーカー / 棚番号 / 初期カウント / 単位 / 最低数 / 品名メモを入力します
  <br>
5. 履歴（出庫 / 入庫 / 棚卸し（実数セット） / 品目登録）
  <br>
　o	出庫履歴と入庫履歴は11件が表示され、超過分は各欄内スクロールで確認できます
  <br>
　o	棚卸し履歴と品目登録履歴は6件が表示され、超過分は各欄内スクロールで確認できます
  <br>


## 使い方

1ページ目
在庫操作（入出庫）
1.	”在庫管理”をブラウザで開く
2.	「棚番号」を選択する
3.	「操作」を選択する（出庫(－) or 入庫(＋)）
4.	「対象品目」を選択する
※同じ製品でもメーカーが異なる場合があるため注意してください
5.	「数量」を入力する(半角数字)
6.	「品目登録」の記入者名を入力する
7.	連絡事項がある場合などは「メモ」に入力してください
※入力したメモは履歴に表示されます
8.	「履歴に記録して反映」で品目登録を実行する
在庫が最低数になった場合は「M1に報告してください」というバナーが表示されるため、M1に速やかに報告して発注を促してください。

在庫操作（棚卸し）
1.	”在庫管理” をブラウザで開く
2.	「棚番号」を選択する
3.	「操作」を選択する（棚卸し（実数セット））
4.	「対象品目」を選択する
※同じ製品でもメーカーが異なる場合があるため注意してください
5.	「数量」を入力する(半角数字)
6.	「品目登録」の記入者名を入力する
7.	連絡事項がある場合などは「メモ」に入力してください
※入力したメモは履歴に表示されます
8.	「履歴に記録して反映」で品目登録を実行する

## 保存先と保存期間

- 保存先: ブラウザの `localStorage`
- 保存単位: 「同じブラウザ + 同じ端末 + 同じURL」ごと
- 保存期間: ユーザーが消去しない限り維持（ブラウザデータ削除で消える）
- 保存量: 1オリジンあたり通常約5MB前後（ブラウザに依存）
- このアプリ内の履歴は最大500件で古い順に削除
