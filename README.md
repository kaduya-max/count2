# 在庫管理サイト

シンプルなブラウザ版の在庫管理ツールです。以下を記録できます。

- 棚卸しのやり方
- 品名 / メーカー / 棚番号 / メモ / 最低数
- 在庫カウント（入庫・出庫・棚卸し）
- 在庫が最低数以下になったときの「M1に報告してください」警告
- 入力履歴（出庫 / 入庫 / 棚卸し（実数セット） / 品目登録を分離）

## 画面構成（タブ）

1. 現在の在庫（在庫操作 + 在庫一覧）
   - 在庫操作と在庫一覧で、棚番号絞り込みをそれぞれ別に設定できます
   - 在庫一覧は棚番号が若い順で表示されます
2. 品目登録
3. 履歴（出庫 / 入庫 / 棚卸し（実数セット） / 品目登録）
   - 出庫履歴と入庫履歴は8件ぶんの高さで固定し、超過分は各欄内スクロールで確認できます
   - 棚卸し履歴と品目登録履歴は5件ぶんの高さで固定し、超過分は各欄内スクロールで確認できます

## 使い方

1. `index.html` をブラウザで開く
2. 1ページ目「在庫操作」の中で記入者名を入力
3. 品目登録で初期在庫と最低数を登録
4. 在庫一覧の「編集」から登録済み品目を更新（編集は履歴に残しません）
5. 在庫操作で入庫・出庫・棚卸しを記録
   - 在庫操作フォームは「棚絞り込み + 対象品目」→「操作 + 数量」→「記入者名」→「記入内容」の順です
6. 在庫が最低数以下になると警告が表示され、×で閉じるまで操作を再開できません
7. 入庫で最低数を上回ると、在庫一覧の文字色は黒に戻ります

## 保存先と保存期間

- 保存先: ブラウザの `localStorage`
- 保存単位: 「同じブラウザ + 同じ端末 + 同じURL」ごと
- 保存期間: ユーザーが消去しない限り維持（ブラウザデータ削除で消える）
- 保存量: 1オリジンあたり通常約5MB前後（ブラウザに依存）
- このアプリ内の履歴は最大500件で古い順に削除
